💡 CONCEPTOS BASICOS:

MOCK:
   Datos ficticios que se usan en pruebas
   Ejemplo: mockGames = [{id: 1, title: 'Game', price: 19.99}]

RENDER:
   Mostrar un componente React dentro de la prueba
   Ejemplo: const { getByText } = render(<MyComponent />)

QUERY:
   Buscar elementos en la página renderizada
   Ejemplo: getByText('Hello') - busca el texto "Hello"

FIRE EVENT:
   Simular una acción del usuario (click, escribir, etc)
   Ejemplo: fireEvent.click(button) - simula un click

EXPECT:
   Verificar que algo es verdadero
   Ejemplo: expect(element).toBeInTheDocument()

AWAIT/WAITFOR:
   Esperar a que algo suceda (para cambios asincronos)
   Ejemplo: await waitFor(() => expect(el).toHaveText('Resultado'))
